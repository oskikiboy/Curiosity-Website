Command Outcome
;Setprefix Set custom prefix for your server.
;Membercount Total amount members on the server.
;Joinlink Gives you a join link to your server got 24 hours.
;Lockdown ;lockdown or ;LD 60 [Sec - Min - Hours - Days ]
This will lock down your channel so no one accept staff in speak in it
;Lockdown Release [ ;LD Release / Unlock ]
This will unlock your channel so everyone can talk again.
;Say #channel msg Say someting in a specific channel or in the same channel.
;Warn @user REASON Warns the specified user for the action defined.
This will be saved in a Database See [Warnings @user] for more info.
;Warnings @user This will show you how many times a user got warned or kicked on your server.
;Kick @user REASON Kicks the specified user from the server.
;Purge ;Purge {Nr of messages} - To delete a specific amount of messages.
;Purge {empty} - To delete 100 messages by default
;Ban @user REASON Bans the specified user from the server.
;Vmute @user Voice mute member server wide.
;Vumute @user Voice Unmute member server wide.
